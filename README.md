## Run locally

1.  Clone repository
  
```bash
  git clone https://github.com/loukik2022/PayMe
```

2.  Install dependencies using npm 

```bash
  cd PayMe
  npm install
  cd client
  npm install
```

3. Run batch file "run PayME.bat" and go to the address shown on the cmd window (ex:  http://localhost:5173/)
